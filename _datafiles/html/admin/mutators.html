Mutator Editor

Identity
Lowercase alphanumeric with hyphens/underscores. Cannot be changed after creation.
Room Title Modifier
How the text is applied to the room title. prepend adds before, append adds after, replace substitutes it entirely. If text is empty with replace, only the color pattern is applied to the existing title.
Text injected into the room title. e.g. "(scorching)" appended makes "Town Square (scorching)".
Applied to the injected text (or the whole title on replace with no text).
Room Description Modifier
prepend inserts a line above the room description, append inserts a line below, replace substitutes the entire description. If text is empty with replace, only the color pattern is applied.
Descriptive text added to the room description while the mutator is active.
Color pattern applied to the injected description text.
Room Alert Modifier
Alerts are special highlighted messages shown below the room description (like "You can train here!"). Mutator alerts are always added to the alert list — they never overwrite existing room alerts.
Alert text shown to players while the mutator is active. Automatically centered if shorter than 65 characters. Leave blank for no alert.
Color pattern applied to the alert text.
Lifecycle & Timing
How long the mutator stays active before it despawns. Accepts durations ("10 minutes", "3 day") or in-game time-of-day events ("sunrise", "sunset", "noon", "midnight"). Leave blank for a permanent mutator.
After decaying, how long until the mutator re-activates. Uses the same format as Decay Rate. Time-of-day events ("noon", "midnight", "sunrise", "sunset") delay initialization until that event occurs. Leave blank if the mutator should not respawn.
When this mutator decays, it transforms into this other mutator instead of simply despawning. This can form chains or circular cycles (e.g. day/night alternation).
Buffs
Buffs continuously applied to all players and their followers while they are in this room and the mutator is active. Applied each round as an "area" buff.
Buffs continuously applied to all mobs in the room while the mutator is active. Applied each round as an "area" buff.
Buffs applied only to mobs that originally spawned in this room (not wandering mobs). Useful for giving "home turf" advantages to native creatures.
Room Effects
Adjusts the room's visibility level. Added to the room's base visibility (0=dark, 1=can see room, 2=can see room and exits). Final value is clamped to 0–2. Negative values can make lit rooms dark; positive values can illuminate dark rooms.
Overrides the room's PvP setting while the mutator is active. Only visible to players when global PvP is set to "limited" mode. Multiple mutators on the same room apply in order — the last one wins.
Tags
or type a custom tag:
Tags applied to the room while this mutator is active. These do not modify the room's permanent tags. Modules and scripts can check for these tags at runtime.
Exits
Temporary exits that appear only while this mutator is active. Applied after normal room exits, so they can override existing exits with the same name. Secret exits are only visible to players who have previously visited the destination room.
